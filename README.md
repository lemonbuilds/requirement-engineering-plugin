# Requirements Engineer Plugin

Lokales Codex-Plugin fuer Requirements Engineering.

## Enthaltene Skills

- `requirements`: Erstellt strukturierte Anforderungskataloge mit funktionalen Anforderungen, nicht-funktionalen Anforderungen und Randbedingungen.
- `user-stories`: Formuliert, prüft und strukturiert User Stories mit Akzeptanzkriterien, Epics und Traceability.
- `use-case-diagram`: Erstellt PlantUML-Use-Case-Diagramme aus Anforderungen.
- `use-case-spec`: Erstellt detaillierte Use-Case-Spezifikationen.

## Struktur

```text
.codex-plugin/plugin.json
skills/
  requirements/
  user-stories/
  use-case-diagram/
  use-case-spec/
```
